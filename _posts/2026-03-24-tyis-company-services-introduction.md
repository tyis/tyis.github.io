# 東洋インターネットサービス株式会社（TOYO）の事業紹介

![TOYOサービス体系]({{ site.url }}/assets/images/svg/toyo-services-overview.svg)

**[東洋インターネットサービス株式会社（TOYO）](https://tyis.co.jp/){:target="_blank"}** は、神奈川県横浜市みなとみらいに拠点を置くIT企業です。

データセンター・クラウドコンサルティングを中心に、ITアウトソーシング、回線サービス、IT資産管理まで、**企業のIT基盤を包括的にサポート**するサービスを展開しています。

## 事業領域

### 1. データセンター / クラウドコンサルティング

企業のITインフラに関する最適な環境構築を提案・支援します。

**主なサービス内容：**
- データセンターの選定・移行支援
- クラウド環境（AWS、Azure、GCP等）の設計・構築
- オンプレミスとクラウドのハイブリッド構成提案
- コスト最適化コンサルティング
- 災害対策（DR）の設計・構築

企業のビジネス要件やセキュリティ要件に応じて、**最適なインフラ環境**をご提案いたします。

### 2. ITアウトソーシング（MSP・BPO）

企業のIT運用を専門チームが代行します。

**MSP（マネージドサービスプロバイダー）：**
- サーバー・ネットワーク機器の24時間監視
- 障害検知・一次対応
- パッチ管理・セキュリティアップデート
- 定期的なシステムレポート

**BPO（ビジネスプロセスアウトソーシング）：**
- ITヘルプデスクの運用
- アカウント管理・権限管理
- IT資産管理の代行
- ベンダー調整・管理

**ITアウトソーシングのメリット：**

| 課題 | 効果 |
|------|------|
| IT人材不足 | 専門チームによる安定運用 |
| 運用コスト増大 | 固定費の変動費化 |
| 属人化リスク | チーム体制による継続性確保 |
| セキュリティ対応 | 最新の脅威への迅速な対応 |

### 3. インターネット回線・国際専用線再販売

企業のネットワーク接続に関するサービスを提供します。

- **法人向けインターネット回線**の手配・提供
- **国際専用線**の再販売・運用支援
- 回線冗長化の提案
- ネットワーク設計・構築

海外拠点との接続や、高品質な回線サービスを必要とする企業様に最適なソリューションをご提供します。

### 4. IT資産管理サービス

IT機器のライフサイクルを一貫してサポートします。

```
導入支援 → 運用管理 → データ消去 → 買取・リサイクル
```

**関連サービス：**
- **[TOYO電子記録抹消サービス](https://erasure.tyis.co.jp/){:target="_blank"}**：専門機器によるデータ完全消去
- **[リサイクルポケット](https://recyclepocket.com/){:target="_blank"}**：IT機器の買取・リサイクル

## 情報セキュリティへの取り組み

### ISO 27001:2022（ISMS）認証取得

弊社は **ISO 27001:2022（ISMS）認証**を取得し、国際基準に準拠した情報セキュリティ体制を構築しています。

```
審査登録機関：BSIグループジャパン株式会社
認証規格：ISO/IEC 27001:2022
認証登録番号：IS 793641
登録範囲：
・データセンター/クラウドコンサルティング
・ITアウトソーシング（MSP・BPO）
・インターネット回線再販売・国際専用線再販売
```

お客様の機密情報を取り扱うすべてのサービスにおいて、ISMSに基づく**厳格なセキュリティ管理**を実施しています。

## TOYOの強み

### 包括的なサービスラインナップ

ITインフラの構築から運用、廃棄まで、**企業のITに関するあらゆる課題**にワンストップで対応できることがTOYOの最大の強みです。

```
┌─────────────────────────────────────────┐
│           TOYOサービス体系               │
├─────────────────────────────────────────┤
│ [構築] データセンター/クラウドコンサル     │
│ [運用] ITアウトソーシング (MSP/BPO)      │
│ [接続] 回線サービス・国際専用線           │
│ [廃棄] データ消去・買取・リサイクル       │
└─────────────────────────────────────────┘
```

### 国際基準のセキュリティ

ISMS認証取得企業として、すべてのサービスにおいて情報セキュリティを最優先に運用しています。

### 横浜みなとみらいの拠点

神奈川県横浜市みなとみらいに拠点を構え、首都圏のお客様を中心にサービスを展開しています。
